Hexo & Github Pages搭建静态博客

网上有许多教程，方法很多,但许多萌新都看不懂云里雾里;用Hexo搭建博客最大的优点就是：轻上传文件快，快访问
也快。因为没有对数据库的操作，所以很安全。当然缺点也是有的，它写文章很慢，如果你用习惯了，速度慢慢会上去的，搭建也比较麻烦。
接下来我这片博客将为大家一步一步解析。
Linux 搭建Hexo的方法：
  • 步骤蛮多的，需要折腾许久
  • 一定要保持耐心，如果错误的地方请仔细再看一边
  • 写hexo博客，需要了解一些markdown的一些语法

    独立的才是自己的
